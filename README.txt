

README – Dissertation Project Files

This repository contains all datasets and notebooks associated with my dissertation. All experiments were executed in Google Colab.

---

1. Anomaly Detection Models & Severity Scoring

ECOD_ProbabilityModel.ipynb

  * Implements the ECOD anomaly detection model.
  * Detects anomalies in the SWaT dataset.
  * Generates XAI outputs that are stored in XAI_Results.zip.
  * Applies the severity scoring framework.

DeepSVDD_NeuralModel.ipynb

  * Runs the DeepSVDD anomaly detection model.
  * Detects anomalies, applies the severity scoring framework, and produces initial XAI outputs.
  * Outputs are stored in `XAI_Results.zip` for further explainability analysis.

---

2. Explainable AI (XAI)

XAI_from_DeepSVDD.ipynb

  * Uses the results in `XAI_Results.zip` from DeepSVDD_NeuralModel.
  * Provides a detailed breakdown of explainability.

XAI_from_ECOD.ipynb

  * Uses the results in `XAI_Results.zip` from ECOD_ProbabilityModel.
  * Provides detailed explainability analysis of ECOD’s outputs.

XAI_Results.zip

  * Contains all the explainability results generated by both ECOD and DeepSVDD models.
  * Includes plots, feature attribution data, and component-level explanations.

---

3. Supporting Analysis

SWaT_EDA.ipynb

  * Exploratory Data Analysis (EDA) of the SWaT dataset.
  * Includes data cleaning, descriptive statistics, and visualisations.
  
---

Execution Order:

1. Run ECOD_ProbabilityModel.ipynb or DeepSVDD_NeuralModel.ipynb (for anomaly detection + severity scoring + initial XAI results).
2. Check `XAI_Results.zip` for stored explainability outputs.
3. Run XAI_from_ECOD.ipynb or XAI_from_DeepSVDD.ipynb for detailed explainability visualisations.
